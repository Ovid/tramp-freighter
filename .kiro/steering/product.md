# Product Overview

Sol Sector Starmap Visualization - An interactive 3D star map for exploring star systems within 20 light-years of Sol.

## Purpose

Transform static astronomical data into an engaging, dynamic 3D user experience for navigating and exploring the local interstellar network.

## Key Features

- Interactive 3D visualization of 117 star systems
- Wormhole connectivity network display
- System information panel with classification, coordinates, and status
- Dynamic depth perception with scaling and opacity
- Sci-fi aesthetic with neon accents and volumetric effects

## Target Users

- Explorers/Researchers needing spatial comprehension of the interstellar network
- Game/Scenario Planners utilizing connectivity and system status for strategic planning
